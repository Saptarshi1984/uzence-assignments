# Uzence App – Frontend Assignment

## ✨ Project Summary

This project is a frontend assignment built using **Next.js** and **Tailwind CSS**, designed to demonstrate clean, responsive UI components and a simple data management interface.

## 📦 Features

- ✅ **Reusable InputField Component**  
  With support for label, placeholder, helper text, error message, multiple states (invalid, loading, disabled), and variants (filled, outlined, ghost).

- ✅ **Dynamic DataTable Component**  
  Displays tabular data with support for column mapping, row selection, empty state, and loading indicator.

- ✅ **Mobile-Responsive Design**  
  Styled with Tailwind CSS for simplicity and adaptability.

## 🛠️ Tech Stack

- [Next.js](https://nextjs.org/) – React Framework
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS
- [TypeScript](https://www.typescriptlang.org/) – Type safety
